# WebApp
Initialized web app project

## Usage
1. Clone this project.
2. Run <code>npm install</code> in Node.js cmd.
3. Run <code>npm run start</code> in Node.js cmd.
4. Open localhost:8080 in browser.

## Default Packages
* webpack
* css-loader
* babel
* babel-polyfill
* react
* reactstrap
* react-router
* axios
* redux
